## Features:

- Time zone configuration
  
- 12h time support:
  - 11:30am #lunch
  - 6:30 pm #home
  - 12pm

- 24h time support:
  - 12:30 #call someone special
  - 22:30 #sleep
  - 11:54", "23 (hours)

- Time span text support:
    - 1h 30m #study
    - 30s
    - 90s #do stuff

```

```
2. Obtain telegram bot token from [BotFather](https://t.me/botfather)
